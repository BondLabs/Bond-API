# Bond API

### New / Update User
#### Request

| Method | URL      |
|:-------|:---------|
|POST    |/api/users|

| Type   | Params   | Values |
|:-------|:---------|:-------|
|HEAD    |auth_key  |string  |
|POST    |id        |int     |
|POST    |name      |string  |
|POST    |email     |string  |
|POST    |phone     |int     |
|POST    |password  |string  |
|POST    |age       |int     |   
|POST    |gender    |string  |
